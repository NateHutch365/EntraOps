**buildNameLookup (private async helper):**
- Declare return type `Map<string, { displayName: string | null; objectType: string | null }>`
- `fs.readdir(PRIVILEGED_EAM_BASE, { recursive: true })` — returns string[] of relative paths on Node 18+
- Filter entries where `String(entry).endsWith('.json')`
- For each: `fs.readFile(path.join(PRIVILEGED_EAM_BASE, String(entry)), 'utf-8')` → parseBomJson → if Array.isArray wrap items, else `[parsed]`
- For each item where `typeof item.ObjectId === 'string'`: `map.set(item.ObjectId.toLowerCase(), { displayName: item.ObjectDisplayName ?? null, objectType: item.ObjectType ?? null })`
- Wrap entire function body in `try { ... } catch { return new Map() }` — handles missing directory gracefully

**GET `/` handler:**
```typescript
router.get('/', async (_req, res, next) => {
  try {
    // Read Global.json — return [] if missing
    let guids: string[] = [];
    try {
      const raw = await fs.readFile(GLOBAL_JSON, 'utf-8');
      const parsed = GlobalFileSchema.safeParse(parseBomJson(raw));
      if (parsed.success) guids = parsed.data[0].ExcludedPrincipalId;
    } catch { /* missing or malformed — return empty */ }

    const lookup = await buildNameLookup();
    const items = guids.map(guid => {
      const entry = lookup.get(guid.toLowerCase());
      return {
        guid,
        displayName: entry?.displayName ?? null,
        objectType: entry?.objectType ?? null,
        resolved: entry !== undefined,
      };
    });
    res.json(items);
  } catch (err) { next(err); }
});
```

**DELETE `/:guid` handler:**
- `UUIDParam.safeParse(req.params.guid)` — return 400 `{error: 'Invalid GUID'}` on failure
- Read Global.json — return 404 `{error: 'Global.json not found'}` if missing
- Parse with GlobalFileSchema — return 500 on parse failure
- Check if guid (case-insensitive match) is in `[0].ExcludedPrincipalId` — return 404 `{error: 'GUID not found in exclusions'}` if absent
- Filter out the guid: `data[0].ExcludedPrincipalId = data[0].ExcludedPrincipalId.filter(g => g.toLowerCase() !== guid.toLowerCase())`
- `atomicWrite(GLOBAL_JSON, JSON.stringify(data, null, 2))`
- `res.status(204).end()`
- Wrap in try/catch, `next(err)`
