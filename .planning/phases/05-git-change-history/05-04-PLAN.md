- Labels: "From" and "To" badges in muted style

**3. Create `gui/client/src/components/history/RawDiffViewer.tsx`:**
- Props: `{ diff: string }`
- Renders diff text inside a `<pre>` block with `overflow-x-auto text-xs font-mono bg-muted rounded-lg p-4`
- Line-by-line coloring:
  - Lines starting with `+` (not `+++`): `bg-green-950/20 text-green-300` (dark mode safe)
  - Lines starting with `-` (not `---`): `bg-red-950/20 text-red-300`
  - Lines starting with `@@`: `text-blue-400 font-semibold`
  - Lines starting with `diff --git`: `text-muted-foreground font-bold`
  - Other lines: default muted text
- Split the diff string by newline, map each line to a `<span className={...}>` inside the `<pre>`
- If diff is empty string: show "No raw diff available" muted message

**1. Create `gui/client/src/pages/ComparePage.tsx`:**

- Read `from` and `to` from URL search params using `nuqs` (`parseAsString`)
- If either missing: show error state "Missing comparison hashes. Go back to History and select two commits."
- Use `useCompare({ from, to })` for data
- Layout:
  ```
  <div className="p-6 space-y-6">
    {/* Back link */}
    <Link to="/history" className="text-sm text-muted-foreground hover:text-foreground inline-flex items-center gap-1">
      <ArrowLeft className="h-4 w-4" /> Back to History
    </Link>

    <h1 className="text-[22px] font-semibold">Compare Commits</h1>

    {/* Header: side-by-side commit metadata */}
    <CommitCompareHeader from={data.from} to={data.to} />

    <Separator />

    {/* Structured changes: RBAC system tabs */}
    {/* Use Tabs component, one tab per affectedSystem */}
    <div>
      <h2 className="text-base font-semibold mb-3">Structured Changes</h2>
      <Tabs defaultValue={data.affectedSystems[0]}>
        <TabsList>
          {data.affectedSystems.map(sys => <TabsTrigger key={sys} value={sys}>{sys}</TabsTrigger>)}
        </TabsList>
        {data.affectedSystems.map(sys => (
          <TabsContent key={sys} value={sys}>
            <ChangeSummary sections={data.changesBySystem[sys]} />
          </TabsContent>
        ))}
      </Tabs>
    </div>

    <Separator />

    {/* Raw diff */}
    <div>
      <h2 className="text-base font-semibold mb-3">Raw Diff</h2>
      <RawDiffViewer diff={data.rawDiff} />
    </div>
  </div>
  ```
- If `data.affectedSystems.length === 0`: show "No PrivilegedEAM/ changes between these commits" instead of the tabs
- Loading: Skeleton placeholders for header and content
- Error: Alert with error message
