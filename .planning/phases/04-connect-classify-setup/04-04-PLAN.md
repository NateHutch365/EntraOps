    - Sidebar footer shows grey dot and "Not connected"

    - Form shows Tenant Name input (pre-populated if EntraOpsConfig.json has TenantName)
    - Auth type radio shows Device Code selected by default; User Interactive option visible

    - On failure: toast appears "Classification failed" with no action button
    - In-card "Go to Dashboard" / "Start Over" buttons appear below terminal after stream ends

    **6. Sidebar indicator**
    - After successful auth, sidebar footer shows green dot + tenant name
    - After "Start Over" / disconnect, sidebar shows grey dot + "Not connected"

    **7. Collapsed sidebar**
    - Collapse the sidebar — connection status dot only is shown (no text)
    - Dot tooltip shows "Connected: {tenant}" or "Not connected" on hover
