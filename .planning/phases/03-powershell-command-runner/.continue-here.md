<decisions_made>

- **`shell: false`** for pwsh spawn — non-negotiable security requirement
- **`z.enum(ALLOWLISTED_CMDLETS)` is the first gate** in POST /run — before isRunning() check
- **`isStopped` module-level flag** — not a closure variable; lets stopCommand() set the outcome from outside runCommand's closure
- **`activeRunId` removed** — was set but never read; fails noUnusedLocals tsc rule
- **Stream tests use direct service calls** for event-shape assertions — supertest SSE body buffering is unreliable
- **History written fire-and-forget** in close handler — does not block SSE stream ending
- **RBAC systems passed as comma-joined string** (`-RbacSystems EntraID,Defender`) to pwsh
</decisions_made>
