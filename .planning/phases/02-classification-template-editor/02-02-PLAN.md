**`gui/client/src/components/templates/DiffDialog.tsx`**

A shared dialog component used by both the template save flow (02-03) and the Global Exclusions save flow (02-04). Import: `diffLines` from `diff` package.

Props interface:
```typescript
interface DiffDialogProps {
  open: boolean;
  title: string;           // e.g. "Preview: Classification_AadResources.json"
  before: string;          // JSON.stringify(original, null, 2)
  after: string;           // JSON.stringify(proposed, null, 2)
  onConfirm: () => Promise<void>;
  onCancel: () => void;
  loading?: boolean;       // shows spinner on Confirm button when true
}
```

Render a shadcn `<Dialog>` with:
- `DialogHeader`: `<DialogTitle>{title}</DialogTitle>`
- `DialogContent`: a `<ScrollArea className="h-[400px]">` containing a `<pre>` block with monospace font. Use `diffLines(before, after)` to get an array of `Change` objects, then render each change's `.value` lines with styling:
  - `added === true` â†’ `bg-green-950 text-green-300` (dark green) 
  - `removed === true` â†’ `bg-red-950 text-red-300` (dark red)
  - neither â†’ `text-muted-foreground`
- `DialogFooter`: "Cancel" button (variant="outline", calls `onCancel`) + "Confirm & Save" button (calls `onConfirm`, disabled when `loading`, shows "Savingâ€¦" text when loading)

Data fetching: `useEffect` triggered when `activeTab` changes (for template tabs). If `templateData[activeTab]` is already loaded, skip fetch. Otherwise: `setLoading(true)` â†’ `fetch('/api/templates/${activeTab}')` â†’ parse JSON â†’ `setTemplateData(prev => ({ ...prev, [activeTab]: data.tiers }))` â†’ `setLoading(false)`.
