## Problem

After a classification run, administrators have no way to review and override individual objects' tier assignments from the GUI. They must manually edit classification config JSON files to adjust tier assignments for specific objects.

## Solution

Post-classification review screen where users can:
- Browse objects flagged by a classification run
- Override individual objects' tier assignments inline
- Save changes back to the classification config files
